== Description ==

A responsive slider for displaying content in WordPress, ideal for creating all sorts of galleries, carousels and scrollers.

Visit our [official product page](https://www.minitek.gr/wordpress/plugins/minitek-slider) to see detailed info on the plugin.

For a live demo, go to [https://demo.minitek.gr/wp/minitek-slider-image-slider-1/](https://demo.minitek.gr/wp/minitek-slider-image-slider-1/).

For more advanced features, you can upgrade to the [Pro version](https://www.minitek.gr/support/documentation/wordpress/minitek-slider/free-vs-pro).

For help setting up and configuring Minitek Slider, you can refer to the [Documentation](https://www.minitek.gr/support/documentation/wordpress/minitek-slider).

If you need support, please use our [Help Center](https://www.minitek.gr/support/forum/wordpress/minitek-slider).
